## Persona
O aplicativo foi pensado para estudantes e pessoas que dividem casa, apartamento ou viagens e precisam organizar contas em grupo de forma mais simples.
## Heurística de Nielsen escolhida
Visibilidade do status do sistema.
Escolhemos essa heurística porque o aplicativo mostra de forma clara os gastos, valores pagos e quem ainda está devendo, ajudando o usuário a entender a situação financeira do grupo.
## Fluxo de tarefa
Cadastro de uma conta de luz dividida entre 3 pessoas:
1. O usuário entra no grupo da casa.
2. Clica em “Adicionar Gasto”.
3. Digita a descrição da conta.
4. Informa o valor total.
5. Escolhe quem realizou o pagamento.
6. Seleciona os participantes da divisão.
7. Salva o gasto.
8. O sistema mostra quanto cada pessoa deve pagar.
